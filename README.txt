Seagate ZBD Iometer Fork
========================

This project is based on the open source Iometer project and was modified to add
support for Zone Block Devices (ZBD). 

Use this fork to test raw ZBD devices. 

Iometer project
===============

The Project Homepage can be found at:
http://www.iometer.org/

The SourceForge.net Services can be found at:
http://sourceforge.net/projects/iometer/

To receive notifications about new versions, you can subscribe at:
http://freshmeat.net/projects/iometer/
